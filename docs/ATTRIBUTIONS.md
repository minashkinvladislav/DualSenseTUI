# Third-Party Assets

## Application icon

The application icon uses the [PS5 Controller Gamepad](https://seeklogo.com/vector-logo/497380/ps5-controller-gamepad)
source image from Seeklogo. The page credits Kadu Leite as its designer.

`resources/macos/Seeklogo-PS5-Controller-Gamepad.png` is retained as the
source asset. `scripts/generate-macos-icon.swift` preserves the original
white background while scaling the controller art into
`resources/macos/DualSenseTUI.icns`.

The source page does not state a redistribution license. Confirm permission
with the rights holder before publishing a release that bundles this icon.
